STAR WARS
SCROLL EFFECT
by Steven Kay

This was done using plain CSS.

The tilt was done using 3D CSS transform (rotate X by 35 degrees). It uses viewport units so should work at any resolution.

Note how the text fades out as it reaches the top. This is done by overlaying a custom gradient from black(0%) to transparent(50%) using a high z-index to place it above the text.

The animation uses @keyframe to change the top position in a linear fashion from 35rem to -85rem; the moves the text gradually up the screen, while the camera stays in the same place.

There's also a very subtle glow effect added using text-shadow.

Works in FF, Chrome and Safari (but Safari is painfully slow)
